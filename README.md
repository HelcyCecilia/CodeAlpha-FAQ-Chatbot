# CodeAlpha FAQ Chatbot

## Description

An AI-powered FAQ Chatbot developed as part of the CodeAlpha AI Internship.

## Features

* Interactive GUI using Tkinter
* Answers frequently asked questions
* Uses NLP techniques (TF-IDF and Cosine Similarity)
* Handles multiple user queries
* User-friendly chat interface

## Technologies Used

* Python
* Tkinter
* Scikit-learn

## How to Run

1. Install Python

2. Install required packages:
   pip install scikit-learn

3. Run:
   python chatbot.py
